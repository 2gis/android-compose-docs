Keys

object Keys

All navigation settings key constants.

Keys are grouped by naming convention:

  • route.car.*, route.pedestrian.*, etc. — route search options per transport type

  • route.* — general route behavior settings

  • sound.* — sound notification settings and per-category toggles

  • map_events.* — on-route display flag toggles

  • map_events.permanent.* — permanent (passed-route) display flag toggles

  • voice.* — voice package selection
